# Image Alt Text Reference — index.html

Update every `<img>` tag on your site so the `alt=""` attribute is descriptive.
Below are suggested alt-text values for each known image in your repo.

## Inline content images (base64 or otherwise)

For each inline image in your content sections, use alt text that describes
both the subject AND the coaching concept the image illustrates. Examples:

- Mindfulness / meditation image  →  alt="Woman in meditation practice — mindfulness and self-care"
- Gears / brain / pause image     →  alt="Illustration of brain gears — pause to process emotions"
- EFT tapping illustration        →  alt="EFT Tapping technique — Emotional Freedom Techniques practice"
- Gratitude image                 →  alt="Journal and pen representing daily gratitude practice"
- Trust / connection image        →  alt="Two people in conversation — building trust in coaching"
- Strength / dumbbell image       →  alt="Dumbbells representing the 'choice muscle' — building resilience"
- Hero's journey image            →  alt="Path through mountains representing the hero's journey"

## How to find/replace in your HTML

Search your `index.html` for `alt=""` and `alt="image"` — these are the
main culprits. Replace each with a descriptive phrase drawn from the table above
or the section heading the image appears under.
